# API 端点
https://api.tikhub.dev/api/v1/douyin/app/v3/fetch_one_video

# 认证方式
Authorization: Bearer <API_KEY>

# 参数
{"aweme_id": "视频 ID"}

### 抖音视频下载流程
1. 使用 `douyin-cli.py category <分类>` 获取视频列表
2. 解析输出提取视频 ID
3. 调用 TikHub API 获取视频信息
4. 从返回的 URL 下载视频文件
5. 保存为 `排名_标题_视频 ID.mp4` 格式
